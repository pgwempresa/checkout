Como encaixar na sua loja HTML

Sua loja continua dona do carrinho, order bumps e da tela. O checkout-pay so vira o motor de pagamento.

1. Inclua o script
Aponte para checkout.seudominio.com/checkout-client.js.
2. Inicialize a loja
Envie storeId, storeName e storeUrl para registrar o heartbeat.
3. Finalize o pedido
Use CheckoutClient.createCheckout com items, customer e shipping.
<script src="https://checkout.seudominio.com/checkout-client.js"></script>
<script>
  CheckoutClient.init({
    baseUrl: "https://checkout.seudominio.com",
    storeId: "minha-loja",
    storeName: "Minha Loja",
    storeUrl: "https://www.minhaloja.com.br"
  });

  async function finalizarCompra(cart, customer) {
    return CheckoutClient.createCheckout({
      referenceId: "PED-" + Date.now(),
      paymentMethod: "pix",
      customer,
      items: cart
    });
  }
</script>
